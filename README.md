# Projet de Stage – Amélioration continue en production

## Contexte
Ce projet a été réalisé lors de mon **stage chez SEBN, Tunisie**.  
L’objectif était d’analyser les flux de production, d’identifier les goulots d’étranglement et d’optimiser les postes de travail afin d’améliorer la productivité globale.

## Objectifs
- Analyser les flux de production pour identifier les points critiques  
- Optimiser les postes de travail pour réduire les goulots d’étranglement  
- Mettre en place des actions Lean et d’amélioration continue  
- Fournir des recommandations basées sur les données et les meilleures pratiques

## Méthodologie
- Analyse des flux avec **VSM (Value Stream Mapping)**  
- Identification des causes racines avec **Ishikawa**  
- Application des méthodes **Lean, Kaizen, 5S**  
- Suivi des améliorations avec **Excel** et **Power BI**  
- Gestion de projet selon la méthodologie **Agile**  

## Outils utilisés
- Lean, Kaizen, 5S, VSM, Ishikawa  
- Agile pour la gestion des actions  
- Excel pour les données et calculs  
- Power BI pour tableaux de bord et visualisations

## Résultats
- Identification et réduction des goulots d’étranglement  
- Optimisation des postes de travail  
- Amélioration mesurable de la productivité  
- Mise en place de recommandations pour une amélioration continue
